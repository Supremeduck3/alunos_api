## 📋 Sobre
Backend RESTful para cadastro e gerenciamento de alunos com Prisma ORM e PostgreSQL.

## ✨ Funcionalidades
- CRUD completo de alunos
- Migrations com Prisma
- Seed de dados

## 🚀 Como Rodar
```bash
git clone https://github.com/Supremeduck3/alunos_api.git
cd alunos_api && npm install
# Configure DATABASE_URL no .env
npx prisma migrate dev --name init
npm run dev
```

## 🛠️ Tecnologias
Node.js • Express • Prisma • PostgreSQL • Nodemon
